# 🐍 Snake Game

一个简洁精美的贪吃蛇网页游戏，支持桌面端和移动端。

## 功能特点

- **经典玩法** — 方向键控制贪吃蛇移动，吃到食物增长并得分
- **加速机制** — 随着得分增加，游戏速度逐步提升（共 5 档速度）
- **最高分记录** — 通过 `localStorage` 保存本地最高分
- **移动端适配** — 支持触摸滑动和虚拟方向键（D-Pad）
- **键盘快捷键** — 方向键 / WASD 控制方向，空格暂停，R 键重新开始
- **视觉反馈** — 游戏状态指示器（准备中/进行中/已暂停/已结束）
- **动态背景** — 浮动光斑动画背景

## 操作方式

| 操作 | 桌面端 | 移动端 |
|------|--------|--------|
| 移动 | 方向键 / WASD | 滑动屏幕 / D-Pad 按钮 |
| 暂停/继续 | 空格键 | — |
| 重新开始 | R 键 | 点击"再来一局" |

## 速度档位

| 得分 | 速度 (ms/帧) |
|------|-------------|
| 0-4  | 150 |
| 5-9  | 130 |
| 10-14 | 110 |
| 15-19 | 90  |
| 20+  | 70  |

## 运行方式

直接用浏览器打开 `snake.html` 即可游玩。

## 技术栈

- 原生 HTML + CSS + JavaScript
- Canvas 绘制游戏画面
- 响应式布局，适配桌面与移动端
